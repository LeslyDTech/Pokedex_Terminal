# Pokémon Terminal

## Overview

This project is a simple Python program that demonstrates the use of object-oriented programming (OOP). It creates a `Pokemon` class with attributes and methods to represent Pokémon and display their information.

## Features

- Creates Pokémon objects using a class.
- Stores Pokémon information such as:
  - Pokédex entry number
  - Name
  - Type(s)
  - Description
  - Level
  - Region
  - Caught status
- Allows each Pokémon to "speak."
- Displays detailed information about each Pokémon.

## Technologies Used

- Python 3

## How to Run

1. Make sure Python 3 is installed.
2. Save the code as `pokemon.py`.
3. Open a terminal or command prompt.
4. Navigate to the project folder.
5. Run the program:

```bash
python pokemon.py
```
